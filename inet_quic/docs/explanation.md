# INET-QUIC Integration

This work extends the INET framework to support QUIC-based communication in a LEO satellite simulation environment.

## Key Points

- QUIC is integrated as a transport layer over UDP
- Supports end-to-end communication across satellite nodes
- Compatible with multi-hop routing in LEO networks

## Purpose

To evaluate QUIC performance under:

- Variable delay
- Packet loss
- Frequent handovers

## Note

This is a simulation-based implementation and focuses on transport behavior rather than full protocol implementation.
